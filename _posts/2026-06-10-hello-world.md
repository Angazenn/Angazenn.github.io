After thinking about it for a while, I've finally set up my personal blog. This site is built with [Jekyll](https://jekyllrb.com/) and hosted on [GitHub Pages](https://pages.github.com/).

## Why Jekyll?

I chose Jekyll for a few reasons:

- **Static and fast** — no database, no server-side processing. Every page is pre-rendered HTML.
- **Markdown** — I can write posts in Markdown, which makes formatting a breeze.
- **Free hosting** — GitHub Pages hosts Jekyll sites for free with a custom domain if you want it.
- **Version control** — my entire blog lives in a Git repository. Every post, every change is tracked.

## What to Expect

I plan to write about:

1. **Software development** — tips, patterns, and lessons learned from building real things
2. **Technology** — interesting tools, frameworks, and ideas I come across
3. **Projects** — deep dives into things I'm building

## Under the Hood

Here's what the code for this site looks like behind the scenes:

```yaml
# _config.yml
title: My Blog
theme: minima
plugins:
  - jekyll-feed
  - jekyll-seo-tag
```

Simple, clean, and effective. Stay tuned for more posts!
